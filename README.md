# Mini Tool-kit

This is the beginning of my coding journey that I will be sharing outside of my class, a multi-purpose tool made with python.

##Installation

This tool does not requires direct installation; it can be run as long as you have an IDE like VSCode on your device.

##Usage

The three built-in functions are:
 A budget tracker
 An even-or-odd checker for numbers
 A grade claculater that gives you your overall average score
